## How to Use It

### Option 1: Test Locally (Easiest)
1. Download both `index.html` and `profile.jpg`
2. Put them in the same folder on your computer
3. Double-click `index.html` to open it in your browser

### Option 2: Deploy to GitHub Pages (Free Hosting)
1. Create a GitHub account (if you don't have one)
2. Create a new repository named `yourusername.github.io`
3. Upload both `index.html` and `profile.jpg` to the repository
4. Your site will be live at `https://yourusername.github.io`

### Option 3: Deploy to Netlify (Free & Easy)
1. Go to [netlify.com](https://netlify.com)
2. Sign up for a free account
3. Drag and drop your folder with both files
4. You'll get a free URL instantly!

### Option 4: Custom Domain
You can use any web hosting service (GoDaddy, Bluehost, HostGator, etc.) and upload these files via FTP.

## Customization Tips

### Change Colors
Look for these color codes in the CSS and replace them:
- `#00A1E0` - Light blue
- `#0070D2` - Salesforce blue
- `#032d60` - Dark blue

### Update Content
- Your email, phone, and LinkedIn are already in the contact section
- To add more certifications, copy a `cert-card` div and update the details
- To add more jobs, copy a `timeline-item` div in the experience section

### Add a Custom Domain
Once deployed, you can point your own domain (like `avinashbollimuntha.com`) to your hosted site.
